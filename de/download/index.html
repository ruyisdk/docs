Ruyi herunterladen​

Ruyi bietet vorgefertigte Binärdateien für die Architekturen x86_64, riscv64 und aarch64 an. Führen Sie uname -m aus, um die Systemarchitektur zu überprüfen, und laden Sie dann die entsprechende Binärdatei herunter.

Architektur auswählen:
$ wget
Nach dem Download führen Sie bitte die folgenden Befehle basierend auf dem tatsächlichen Dateinamen im obigen Link aus (Befehle werden automatisch mit der Architektur aktualisiert):
$ chmod +x ./ruyi
$ sudo cp -v ruyi /usr/local/bin/ruyi
tip

Falls die Seite den Download-Link nicht korrekt anzeigt, können Sie das RuyiSDK-Mirror besuchen, um die neuesten Binärdateien herunterzuladen. Alternativ finden Sie sie auch bei den GitHub-Releases.

RuyiSDK-Paketmanager installieren​

Der obige Befehl wurde entsprechend Ihrer Architektur automatisch generiert. Stellen Sie sicher, dass die Binärdatei als ruyi installiert ist und sich in einem Verzeichnis befindet, das im PATH enthalten ist.

tip

Stellen Sie sicher, dass der Dateiname ruyi lautet.

Führen Sie den folgenden Befehl aus, um die Installation zu überprüfen:

bash
$ ruyi version

Wenn der Befehl die Versions- und Copyrightinformationen ausgibt, wurde Ruyi erfolgreich installiert. Falls der Befehl fehlschlägt, überprüfen Sie Ihre Systemumgebung und wiederholen Sie die Schritte.
